Format of Multiple Swimmers File

The format of the file should be a text file in the following format:

TEAM:Full Team Name 1
FirstName LastName
FirstName LastName
...
TEAM:Full Team Name 2
FirstName LastName
FirstName LastName
...
etc.

Example:
TEAM:Naval Academy Aquatic Club
David Thornton
John Doe
TEAM:University of Florida
Caeleb Dressel
Jane Doe
